Visão Geral
----------

LNEGC é uma linguagem controlada em português do Brasil projetada para facilitar
a descrição estruturada de algoritmos, estruturas de dados e requisitos de software,
fornecendo uma interface padronizada entre desenvolvedores e sistemas de IA para
geração automática de código.

Características Principais
-----------------------

* Suporte para múltiplas linguagens de programação
* Documentação automática
* Testes automatizados
* Validação de regras de negócio
* Integração com sistemas de IA
* Formato estruturado e padronizado
* Sintaxe em português do Brasil
